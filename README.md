# SpaceFighter
Tower Defense Game using Java OOP



https://github.com/tenizz/SpaceFighter/assets/41874992/4e5c708f-232d-4d2c-ac93-956ab3b64a04




## Description

   There are Asteroids and Comets going to the Earth which you are required to protect. You are given certain Satellites which can destoy Asteroids and Comets. Also, you can buy Heal Box which gives you more Cities (HP).
